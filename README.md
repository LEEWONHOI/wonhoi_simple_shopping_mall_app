## 🛠️ 사용 라이브러리

- Room
- Koin
- LiveData , ViewModel
- Coroutines
- Retrofit2, OkHttp3
- Glide
- Firebase

<br/>
<br/>


## 📱 구현한 기능

- MVVM 아키텍처 방식으로 구현
- Mock 형태로 만든 임시 제품 리스트 화면과 프로필 화면
- 클릭 시 상품의 상세 정보 및 주문기능
- 파이어베이스 토큰을 활용해서 구글 로그인 기능
- 구글 프로필 확인 및 주문한 리스트 출력
